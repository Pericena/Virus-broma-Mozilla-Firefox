- Problemas en Firefox provocados por malware
Malware es la abreviación de “Malicious Software” (software malintencionado), un término que se usa generalmente para referirse al software instalado en tu equipo que está diseñado para infiltrarse o dañar un equipo informático sin el conocimiento de su propietario. A veces un problema con Firefox puede deberse a que este tipo de software se haya instalado en tu equipo sin que te hayas dado cuenta. Este artículo describe los síntomas más comunes que presenta el software malintencionado y cómo conseguir que no se instale y deshacerse de él.
